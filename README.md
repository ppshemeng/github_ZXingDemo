github_ZXingDemo
================

二维码条码扫描（控件版）

将Google的开源项目ZXing封装成控件，便于在项目中引用。

本项目有两个工程，ZXingScanner Demo为测试项目，ZXingScanner (lib)为控件源码。

在测试项目中引用的是编译好的jar包，如果你比较懒的话就直接用吧。
